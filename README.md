# 🎮 Pokémon Arena

Une application web combinant un Pokédex interactif et un système de combat Pokémon, développée dans le cadre d'un projet à la Wild Code School.

## 📋 Description

Pokémon Arena est une application web qui permet aux utilisateurs d'explorer l'univers Pokémon à travers un Pokédex complet et de participer à des combats Pokémon dynamiques. Le projet a été réalisé sur trois sprints, mettant l'accent sur une expérience utilisateur fluide et une interface responsive.

## ✨ Fonctionnalités

### Pokédex Interactif
- Base de données complète des Pokémon
- Système de recherche avancé
- Filtres multiples pour une navigation optimisée
- Fiches détaillées pour chaque Pokémon

### Système de Combat
- Combats Pokémon en temps réel
- Sélection stratégique des Pokémon
- Système d'attaques et de défense
- Animations dynamiques

### Interface Utilisateur
- Design responsive adapté à tous les écrans
- Navigation intuitive
- Interface moderne et attrayante
- Expérience utilisateur optimisée

## 🛠️ Technologies Utilisées

- **Frontend:**
  - React.js
  - CSS
  - JavaScript
- **Backend:**
  - Express.js
  - Node.js
- **API:**
  - PokéAPI

## 🚀 Installation

```bash
# Cloner le repository
git clone https://github.com/votre-username/pokemon-arena.git

# Installer les dépendances
cd pokemon-arena
npm install

# Lancer l'application
npm start
```
